  • 我应当选择哪种计价方案？

    开发版：开发版让用户可以在开发阶段和个人项目中免费使用 LeanCloud 的大部分功能。大部分商业应用在发布给外部用户后会超过开发版的用量限制，将会需要升级到商用版。

    商用版：商用版是高性能、高可用的商业应用解决方案，可以满足大部分线上产品的需求。商用版集群统一进行容量规划和扩容，服务目标是流量相对比较稳定，逐渐增长的应用。为了避免因个别应用流量突增或受到攻击而影响所在集群的稳定性，商用版应用仍然有并发请求数的软限制。在合理使用的情况下，这个限制可以通过提交工单提高。但如果您的应用如果因为推广或业务性质等原因会经常出现流量的大幅波动，建议您选择企业版来预留充足的资源。

    企业版：企业版是可以灵活应对多样化需求的企业级解决方案，主要有两种部署方式：独立集群和私有集群。独立集群是指把部分或全部服务部署在 LeanCloud 提供的服务器集群上，使用体验和商用版一致，但由客户确定集群容量。独立集群特别适用于预期产品发布后在短期内会快速增长，需要根据自己的判断保证充足资源的客户。私有集群指把 LeanCloud 的服务部署在客户自己的服务器集群上，LeanCloud 的软件系统仍然由 LeanCloud 提供运维服务。这种方式主要适用于需要与在自己机房的其它系统进行集成，或对数据存储地有特定需求的客户。

  • 我可以随时在不同计价方案间切换吗？

    由于我们会对商用版应用和开发版应用进行物理隔离，存在数据迁移过程和硬件成本，因此在升级到商用版 30 天之后才可以降级到开发版。这一规则只对用户主动切换定价方案的应用生效，如果是我们以某一种规则自动将您的应用升级 / 降级，则不受此规则约束。

  • 哪些功能是收费的？哪些是免费的？

    我们的收费项目包括数据存储、文件存储和 CDN、实时聊天、短信，以及云引擎实例。针对开发版应用，我们也提供了较宽裕的免费额度。除此之外，我们还提供很多免费功能，包括推送通知、用户管理、社交关系管理等。

  • 商用版的最低消费是什么意思？

    商用版的最低消费用于覆盖我们对商用版应用提供技术支持和运维保障的费用。对于每个商用版应用，如果当天各项服务的消费总和（文件流量和文件存储空间除外，按月扣费的特殊计费项目除外）不足商用版的最低消费额，我们会按照最低消费额扣除费用，如果超过了最低消费额，我们按照实际消费的金额扣除费用。

  • 即时通讯服务里面的每日登录用户如何计算？一个用户每天登录很多次算一个还是多个？

    同一个用户每天多次登录只要使用的是同样的 ID 就会按照一个用户计算。

  • 数据存储的请求数如何计算？

    我们在各个平台的 SDK 底层都是通过调用 REST API 来实现各功能的，用于计费的就是对 REST API 的请求次数。通常来说每次对数据的新增、保存、查询就是一次请求，但也有一部分 SDK 的批量操作或多步操作会引起多次 API 请求。

  • 我可以得到怎样的技术支持？

    对于所有用户，我们都建议优先选择社区支持。许多开发者都在 LeanCloud 开发者论坛 留下了他们的经验，您可以通过在社区搜索问题快速得到大多数问题的答案。我们的工程师也会在工作时间回答社区中优先级较高的一些问题。

    除此以外，我们也对商用版用户提供一对一的工单技术支持。

  • 工作线程数是什么意思？我需要多少工作线程数？

    工作线程数指的是服务端可同时处理的请求数量。绝大多数应用需要的服务端工作线程数都不会超过 3 个。

    假设服务端响应时间是 20 毫秒，那么每个工作线程每秒钟可以处理 1,000 / 20 = 50 个请求。

    一个日活一万的应用，如果每个用户都产生 50 个请求，而且这一万个用户每天都集中在两个小时内打开应用，这两小时内平均每秒钟发生的请求数就是 10,000 × 50 / 2 / 60 / 60 = 69 个，只需要 69 / 50 = 1.15 个工作线程就足以应对这两个小时的请求高峰。

    考虑到可能出现的峰值，实际运行的产品应当尽量预留一定的冗余。商用版应用默认拥有 60 个工作线程，足以应对绝大部分需求，如有需要也可以按需调整。

  • 开发版应用资源使用量超过限制之后会发生什么？

    数据 API 每天累计请求超过 30,000 次后，后续的请求会无法得到数据，并返回错误码 429，这个限制每日重置。

    服务端工作线程数 超过 3 时，后续的请求会无法得到数据，并返回错误码 429，这个限制在工作线程数小于 3 时解除。

    即时通讯服务 每天累计在线用户超过 500 人后，客户端将无法与 LeanCloud 建立新的连接，这个限制每日重置。

    短信验证服务 和 文件存储服务 均为根据实际使用量单独计费，开发版应用可以不受限制地使用短信和文件存储功能。
