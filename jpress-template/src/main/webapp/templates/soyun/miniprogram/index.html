微信小程序解决方案

省掉域名备案、数据接口开发、后端系统搭建与运维等繁重工作，让小程序开发更简单，效率更高。

用户账户管理

内建账户系统，支持使用微信用户身份一键登录。

除微信提供的基本信息之外，账户系统还支持保存手机号、收货地址等额外信息。

支持微信 UnionId 登录，允许在多个平台、应用中打通用户账号体系。

小程序数据存储

一站式数据管理平台，支持结构化数据的增删改查，自动创建索引优化查询效率。

支持存储资源文件、用户相册、商品图片等二进制文件，默认提供 CDN 加速。

全面支持 HTTPS 协议，2048 位 RSA 密钥加密，确保数据传输安全可靠。

多副本分布式存储，定期备份，99.9999% 数据可靠性，确保万无一失。

云函数与触发器

与 LeanCloud 云引擎服务无缝衔接，提供默认的独立访问域名。

使用云函数快速实现一些在服务端执行的逻辑，比如处理敏感信息、数据聚合和采集等。

提供用户下单、支付的云引擎模版，支持一键部署快速上线。

优势和特色

数据与其他平台互通

让小程序与独立应用共享底层数据，多平台互联互通，让产品和用户在微信生态之外产生价值。

专业支持安全可靠

原生支持 HTTPS / WSS，配备 BGP 高防、域名智能解析及防劫持等多重安全防护体系，为小程序保驾护航。

丰富的 Demo 和 SDK 支持

根据业务场景提供多种 Demo，丰富的 SDK 支持主流开发方案，包括原生开发者工具、wepy、mpvue 等。
